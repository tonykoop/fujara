# Fujara B7 sidecar: bore / side-branch relation starter

This folder is a **sidecar** packet for `fujara` issue `#5` and is intentionally additive
because draft PR `#6` already owns `build/packet/*`.

## Purpose
- Add a small Wolfram starter that compares candidate side-branch geometries against
  the coupling-to-main-bore sensitivity metric.
- Keep the model explicitly marked as a starter and avoid claims of final physical
  validation.
- Produce committed runtime evidence with `wolframscript` output files.

## Files
- `fujara_sidebranch_starter.wl` — starter with assumptions and sensitivity equations.
- `fujara_sidebranch_study.csv` — runtime table generated by the script.
- `fujara_sidebranch_summary.json` — computed peak statistics per case.

## Run command
```bash
wolframscript -file build/wolfram/side-branch-relation/fujara_sidebranch_starter.wl
```

The script writes output files in the same folder and prints the generated paths.

## Notes and assumptions
- This lane is for a **runtime evidence starter** only.
- The formulas are intentionally simplified and labeled runtime-only.
- Geometric coupling labels are not treated as fabrication or acoustic build certification.
